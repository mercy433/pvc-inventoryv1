PVC库存管理 PWA 发布包

文件说明
- index.html：网页入口文件
- manifest.json：PWA 清单
- sw.js：离线缓存脚本
- icons/：主屏幕图标
- .nojekyll：关闭 GitHub Pages 的 Jekyll 处理

GitHub Pages 发布步骤
1. 在 GitHub 新建一个仓库。
2. 把这个文件夹里的所有文件上传到仓库根目录。
3. 在仓库 Settings -> Pages 中开启 GitHub Pages。
4. 选择从默认分支根目录发布。
5. 等几分钟后，用生成的网址访问。
6. 在 iPhone 上用 Safari 打开网址，选择“添加到主屏幕”。

提示
- PWA 安装更推荐用 iPhone Safari 打开后添加到主屏幕。
- 你当前的数据如果要迁移到线上版本，可以先在电脑网页里点“导出数据”，上线后再“导入数据”。
